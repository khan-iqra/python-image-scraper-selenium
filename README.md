# Python Image Scraper using Flask & Selenium

## 📌 Project Overview

This project is a web-based image scraper developed using **Python, Flask, Selenium, BeautifulSoup, and HTML/CSS**. It allows users to search for images by entering a keyword. The application scrapes image results from the web and displays them on a user-friendly interface.

## ✨ Features

* Search images using keywords.
* Web scraping using Selenium.
* Displays multiple image results.
* Responsive Flask web application.
* Simple and user-friendly interface.

## 🛠️ Technologies Used

* Python
* Flask
* Selenium
* BeautifulSoup (bs4)
* Requests
* HTML
* CSS
* JavaScript (if applicable)

## 🚀 How to Run the Project

1. Clone the repository.
2. Install the required packages:

   ```
   pip install -r requirements.txt
   ```
3. Run the application:

   ```
   python app.py
   ```
4. Open the application in your browser:

   ```
   http://127.0.0.1:8000
   ```
